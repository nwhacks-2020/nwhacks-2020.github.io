Rebuild is an Android application which creates a mesh network to automatically share vital disaster information data between nearby devices without Wi-Fi or cell service..

The project was created during January 11-12 for nwHacks 2020, an annual hackathon hosted by nwPlus at the University of British Columbia.

Web Application (Angular & Bootstrap)

The front end of the web application is built using Angular, to create the single-page application with styling and scripting, and Bootstrap to provide styling support. These provided a strong and flexible platform on which to build the application.

Graphic Design (Adobe XD/Figma/Zeplin)

To visually understand how users would navigate and use the application, graphic designs were created using Adobe XD and passed to the developers.

Server (Node.js & Express)

The back end server was written in Node.js with the Express framework. It provides data storage, processing, and communication in tandem with the front end component.

The backend server was written in Golang. It provides data storage, processing, and communication in tandem with the front end component.
